Индексация, срезы и маски

До выполнения индексации назовите смысл выбранных осей, предскажите shape результата и решите, должен ли он разделять память с источником.

Тип: Build Треки: Core Пререквизит: 02-numpy/03-dtypes Время: ~75 минут Результат: по смыслу осей выбирает элементы, строки и столбцы, предсказывает shape результата и явно контролирует изменение источника или независимой копии.

Цели обучения

  • читать положительные и отрицательные индексы, а также срезы start:stop:step;
  • выбирать данные по нескольким осям и предсказывать форму результата;
  • понимать, когда целочисленный индекс удаляет ось, а срез сохраняет ее;
  • строить маски из нескольких условий с &, |, ~ и явной политикой нечисловых значений;
  • различать маску строк, маску столбцов и маску отдельных элементов;
  • отличать basic indexing от advanced indexing и проверять совместное использование памяти;
  • различать чтение через индекс и присваивание в индексируемую область;
  • делать выбор между изменением исходного массива и работой с независимой копией явным.

Проблема

После прошлых уроков мы умеем описать массив через предметные оси, проверить shape и выбрать подходящий dtype. Теперь нужно работать не со всем массивом, а только с нужными наблюдениями и признаками.

Представим маленькую числовую выгрузку заказов:

python
import numpy as np

orders = np.array(
    [
        [101, 1, 1200, 2],
        [102, 3, 4500, 7],
        [103, 2, np.nan, 4],
        [104, 2, 3200, 3],
        [105, 5, 8000, 1],
    ],
    dtype=np.float64,
)

assert orders.shape == (5, 4)

Предметный контракт осей:

text
axis 0: observation = order
axis 1: feature

feature 0: order_id
feature 1: item_count
feature 2: amount
feature 3: delivery_days

order_id здесь тоже хранится числом только ради компактного учебного примера. В реальной таблице идентификатор, сумма и срок доставки имеют разные семантические типы, поэтому после знакомства с pandas такой набор удобнее хранить в DataFrame.

Задача аналитика:

  1. оставить заказы с конечной суммой от 1000 до 5000 включительно;
  2. оставить среди них заказы, доставленные не более чем за 5 дней;
  3. вернуть только order_id, amount и delivery_days;
  4. не изменить исходный массив;
  5. получить результат формы (selected_order, selected_feature).

Здесь легко допустить несколько разных ошибок.

python
sample = orders[:2]
sample[:, 2] = 0

sample выглядит как отдельный массив, но базовый срез обычно является view. Изменение попадет и в orders.

Другой пример:

python
amount = orders[:, 2]

Значения выбраны правильно, но ось признаков исчезла: форма стала (5,), а не (5, 1). Следующая функция может ожидать матрицу наблюдений и признаков и отклонить такой результат.

Поэтому индексация отвечает сразу на три вопроса:

  1. Какие значения выбраны?
  2. Какие оси остались и каков shape?
  3. Разделяет ли результат память с источником?

Ответ только на первый вопрос недостаточен.

Концепция

Индекс — это координата на предметной оси

В одномерном массиве одна позиция индекса относится к единственной оси:

python
values = np.array([10, 20, 30, 40])

assert values[0] == 10
assert values[2] == 30
assert values[-1] == 40
assert values[-2] == 30

Нумерация слева начинается с 0. Отрицательные индексы считаются справа: -1 — последний элемент, -2 — предпоследний.

Индекс должен попадать в длину оси:

python
values[4]   # IndexError: ось имеет длину 4
values[-5]  # IndexError по той же причине

Число в квадратных скобках — не идентификатор наблюдения. values[2] означает третью позицию массива, а не строку с бизнес-ключом 2. NumPy использует позиционную модель.

Срез описывает полуинтервал позиций

Синтаксис среза:

text
start:stop:step

Левая граница включается, правая не включается:

python
values = np.array([10, 20, 30, 40, 50])

np.testing.assert_array_equal(values[1:4], [20, 30, 40])

Такой полуинтервал удобен: длина простого среза start:stop равна stop - start, если обе границы находятся внутри массива и шаг равен единице.

Пропущенная граница означает начало или конец:

python
np.testing.assert_array_equal(values[:3], [10, 20, 30])
np.testing.assert_array_equal(values[2:], [30, 40, 50])
np.testing.assert_array_equal(values[:], values)

Шаг выбирает каждую step-ю позицию:

python
np.testing.assert_array_equal(values[::2], [10, 30, 50])
np.testing.assert_array_equal(values[::-1], [50, 40, 30, 20, 10])

Срез обычно не сообщает об ошибке, если граница вышла за длину массива:

python
np.testing.assert_array_equal(values[:100], values)

Это удобно, но способно скрыть неверное ожидание размера. Если алгоритму нужны ровно три наблюдения, проверьте форму результата отдельно.

В многомерном массиве каждая позиция относится к своей оси

Для матрицы (observation, feature) первый индекс выбирает наблюдения, второй — признаки:

python
assert orders[0, 2] == 1200

Читаем слева направо:

text
orders[0, 2]
       │  └── feature 2: amount
       └───── observation 0: первый заказ

Двоеточие означает «вся ось»:

python
first_order = orders[0, :]
all_amounts = orders[:, 2]

У first_order форма (4,), у all_amounts — (5,).

Целочисленный индекс удаляет ось, срез сохраняет

Это главное правило предсказания формы в базовой индексации.

Выражение Предметный смысл shape
orders[1, 2] одно значение одного признака ()
orders[1] все признаки одного заказа (4,)
orders[1:2] один заказ как матрица (1, 4)
orders[:, 2] сумма каждого заказа как вектор (5,)
orders[:, 2:3] сумма каждого заказа как матрица с одним признаком (5, 1)
orders[:3, 1:4] три заказа и три признака (3, 3)

Проверим различие кодом:

python
assert orders[1].shape == (4,)
assert orders[1:2].shape == (1, 4)

assert orders[:, 2].shape == (5,)
assert orders[:, 2:3].shape == (5, 1)

Значения могут выглядеть одинаково при печати, но контракт для следующей операции разный. Если требуется сохранить ось, используйте срез длины один, а не целое число.

np.newaxis добавляет ось длины один

Предыдущий урок вводил np.expand_dims. Внутри квадратных скобок та же операция часто записывается через np.newaxis:

python
amount_vector = orders[:, 2]
amount_column = amount_vector[:, np.newaxis]

assert amount_vector.shape == (5,)
assert amount_column.shape == (5, 1)

np.newaxis является читаемым псевдонимом для None:

python
same_column = amount_vector[:, None]
assert same_column.shape == (5, 1)

В учебных материалах будем предпочитать np.newaxis, потому что оно явно называет намерение.

Многоточие ... подставляет столько полных срезов :, сколько нужно для оставшихся осей:

python
daily_metrics = np.zeros((2, 3, 2))
with_scenario = daily_metrics[..., np.newaxis]

assert with_scenario.shape == (2, 3, 2, 1)

Это продолжает пример (store, day, metric) прошлого урока: все существующие оси сохраняются, а справа появляется scenario длины один.

Basic indexing обычно возвращает view

К basic indexing относятся целые индексы, срезы, ... и np.newaxis. Когда результат остается массивом, он обычно обращается к тому же буферу данных:

python
source = np.arange(12).reshape(3, 4)
part = source[1:3, :2]

assert np.shares_memory(source, part)

part имеет свои shape и strides, но его значения лежат в буфере source.

python
part[0, 0] = 999
assert source[1, 0] == 999

Такой результат называется view. Он экономит память и позволяет быстро работать с частью большого массива. Проблема возникает не в самом view, а в неявном предположении, что перед нами независимая копия.

Есть и обратная сторона экономии: маленький долгоживущий view продолжает ссылаться на буфер большого источника. Если из большого временного массива нужна только небольшая часть, которую предстоит хранить долго, явная .copy() может освободить исходный буфер раньше. Проверяйте отношение через np.shares_memory; поле .base является деталью цепочки представлений, а не удобным контрактом независимости.

Advanced indexing возвращает копию при чтении

Массив целых индексов выбирает позиции в указанном порядке:

python
source = np.array([10, 20, 30, 40])
selected = source[[3, 1, 1]]

np.testing.assert_array_equal(selected, [40, 20, 20])
assert not np.shares_memory(source, selected)

Позиции можно переставлять и повторять. Для результата приходится собирать новый буфер, поэтому такое чтение относится к advanced indexing и возвращает копию.

Булева маска тоже относится к advanced indexing:

python
mask = np.array([False, True, True, False])
selected = source[mask]

np.testing.assert_array_equal(selected, [20, 30])
assert not np.shares_memory(source, selected)

Минимальная рабочая модель:

Способ чтения Пример Обычный результат
Basic indexing x[1:4], x[:, 2] view, если результат остается массивом
Integer advanced indexing x[[0, 3]] копия
Boolean advanced indexing x[x > 0] копия

Слово «обычный» важно для basic indexing: не пытайтесь угадывать память по внешнему виду. Если независимость существенна, вызовите .copy() или проверьте np.shares_memory.

Сравнение строит булев массив той же формы

python
amounts = orders[:, 2]
amount_ok = amounts >= 1000

assert amount_ok.dtype == np.dtype("bool")
assert amount_ok.shape == amounts.shape == (5,)

Каждое значение маски относится к позиции с тем же номером:

text
amounts:   [1200, 4500, NaN, 3200, 8000]
amount_ok: [True, True, False, True, True]

Сравнение с NaN дает False, но не стоит строить на этом неявную политику. В числовом фильтре лучше прямо назвать требование конечности:

python
amount_is_finite = np.isfinite(amounts)

np.isfinite исключает и NaN, и положительную или отрицательную бесконечность.

Условия объединяются поэлементно

Для массивов используются:

  • & — поэлементное «И»;
  • | — поэлементное «ИЛИ»;
  • ~ — поэлементное «НЕ».

Например:

python
invalid_amount = ~np.isfinite(amounts)
suspicious_amount = (amounts < 0) | (amounts > 100_000)

Каждое сравнение заключайте в скобки:

python
amount_ok = (
    np.isfinite(amounts)
    & (amounts >= 1000)
    & (amounts <= 5000)
)

Условие на другой признак:

python
delivery_days = orders[:, 3]
delivered_quickly = np.isfinite(delivery_days) & (delivery_days <= 5)

row_mask = amount_ok & delivered_quickly

Операторы Python and, or и not ожидают одно логическое значение. У массива значений несколько, поэтому amount_ok and delivered_quickly приводит к ошибке о неоднозначном truth value.

Форма маски определяет, что она выбирает

Для матрицы формы (5, 4) встречаются три разных контракта.

Маска строк имеет форму (5,) и применяется к первой оси:

python
row_mask = np.array([True, False, False, True, False])
selected_rows = orders[row_mask]

assert selected_rows.shape == (2, 4)

Маска столбцов имеет форму (4,) и явно ставится на ось признаков:

python
column_mask = np.array([True, False, True, True])
selected_columns = orders[:, column_mask]

assert selected_columns.shape == (5, 3)

Маска элементов имеет ту же форму (5, 4), что и матрица. При чтении она собирает все позиции True в один одномерный результат:

python
element_mask = orders > 3000
large_values = orders[element_mask]

assert large_values.ndim == 1
assert large_values.size == int(element_mask.sum())

Это не «фильтрация строк». Предметная структура осей в таком результате потеряна.

Маска формы (5, 1) не становится маской строк автоматически. Boolean indexing не обязан транслировать маску так, как арифметические операции используют broadcasting. Для выбора строк нужен контракт ровно (5,).

Чтение и присваивание — разные операции

При чтении advanced indexing создает копию:

python
source = np.array([10, 20, 30, 40])
mask = source >= 30
selected = source[mask]

selected[:] = 0
np.testing.assert_array_equal(source, [10, 20, 30, 40])

Но индексированное присваивание изменяет позиции самого объекта слева:

python
source[mask] = 0
np.testing.assert_array_equal(source, [10, 20, 0, 0])

NumPy не обязан сначала возвращать временную копию source[mask]. Вся конструкция source[mask] = 0 означает «записать значение в выбранные позиции source».

Отсюда следует опасность цепочки:

python
matrix = np.arange(12).reshape(3, 4)
matrix[[0, 2]][:, 1] = -1

Первое чтение matrix[[0, 2]] создает временную копию. Изменится эта копия, а matrix останется прежней. Для изменения источника сформируйте один индекс или явную маску и присвойте через него:

python
matrix[[0, 2], 1] = -1

Два advanced-индекса могут выбирать пары, а не прямоугольник

python
matrix = np.arange(12).reshape(3, 4)
row_indices = np.array([0, 2])
column_indices = np.array([1, 3])

pairs = matrix[row_indices, column_indices]

pairs содержит координаты (0, 1) и (2, 3), поэтому его форма (2,). Это не две строки на два столбца.

Для прямоугольника используйте явный декартов выбор:

python
rectangle = matrix[np.ix_(row_indices, column_indices)]
assert rectangle.shape == (2, 2)

Общие правила совместного advanced indexing значительно шире. Для аналитического минимума достаточно помнить этот failure mode, использовать np.ix_ для прямоугольника или делить сложную выборку на два именованных шага.

Контракт выборки состоит из семи решений

Перед рабочей индексацией ответьте:

  1. Что означает каждая ось источника?
  2. Какие позиции или условия выбирают каждую ось?
  3. Каков ожидаемый shape результата?
  4. Должна ли сохраниться ось длины один?
  5. Что делать с NaN и бесконечностями?
  6. Нужна ли связь с буфером источника или независимая копия?
  7. Если значения изменяются, должен ли измениться источник и подходит ли replacement его dtype?

Такой контракт соединяет три предыдущих урока: смысл осей, форму и тип данных.

Соберите это

Сначала реализуем прозрачный механизм выбора без NumPy. Он покажет, что маска строк — это одно решение True или False на каждое наблюдение, а список столбцов — позиционная проекция признаков.

Шаг 1. Зафиксируйте строки, маску и столбцы

python
rows = [
    [101, 1, 1200, 2],
    [102, 3, 4500, 7],
    [103, 2, None, 4],
    [104, 2, 3200, 3],
    [105, 5, 8000, 1],
]

amount_ok = [True, True, False, True, False]
delivered_quickly = [True, False, True, True, True]
row_mask = [
    amount and delivery
    for amount, delivery in zip(amount_ok, delivered_quickly, strict=True)
]

column_indices = [0, 2, 3]

Проверьте контракт до выборки:

python
assert row_mask == [True, False, False, True, False]
assert sum(row_mask) == 2
assert len(column_indices) == 3

Ожидаемая форма результата — (2, 3).

Шаг 2. Выберите строки и столбцы вручную

python
def select_rows_and_columns(rows, row_mask, column_indices):
    if not rows:
        raise ValueError("rows must not be empty")

    width = len(rows[0])
    if width == 0 or any(len(row) != width for row in rows):
        raise ValueError("rows must form a non-empty rectangle")
    if len(row_mask) != len(rows):
        raise ValueError("row mask must contain one decision per row")
    if any(type(keep) is not bool for keep in row_mask):
        raise TypeError("row mask must contain booleans")
    if not column_indices:
        raise ValueError("at least one column is required")
    if any(column < 0 or column >= width for column in column_indices):
        raise IndexError("column index is out of bounds")

    selected = []
    for row, keep in zip(rows, row_mask, strict=True):
        if keep:
            selected.append([row[column] for column in column_indices])
    return selected
python
manual_result = select_rows_and_columns(rows, row_mask, column_indices)

assert manual_result == [
    [101, 1200, 2],
    [104, 3200, 3],
]

Функция не вычисляет аналитические условия. Она делает видимым механизм:

  • row_mask[i] принимает или отклоняет строку rows[i];
  • порядок выбранных строк сохраняется;
  • каждый результат содержит столбцы в порядке column_indices;
  • создаются новые внутренние списки, поэтому источник не изменяется.

Шаг 3. Проверьте инварианты результата

python
assert len(manual_result) == sum(row_mask)
assert all(len(row) == len(column_indices) for row in manual_result)
assert rows[0] == [101, 1, 1200, 2]

В NumPy цикл выбора скрыт внутри advanced indexing, но предметные инварианты остаются теми же.

Используйте это

Постройте маски отдельно и назовите их

python
import numpy as np

ORDER_ID = 0
ITEM_COUNT = 1
AMOUNT = 2
DELIVERY_DAYS = 3

orders = np.array(
    [
        [101, 1, 1200, 2],
        [102, 3, 4500, 7],
        [103, 2, np.nan, 4],
        [104, 2, 3200, 3],
        [105, 5, 8000, 1],
    ],
    dtype=np.float64,
)

amounts = orders[:, AMOUNT]
delivery_days = orders[:, DELIVERY_DAYS]

amount_ok = (
    np.isfinite(amounts)
    & (amounts >= 1000)
    & (amounts <= 5000)
)
delivered_quickly = np.isfinite(delivery_days) & (delivery_days <= 5)
row_mask = amount_ok & delivered_quickly

Именованные условия легче проверить по отдельности, чем одно длинное выражение:

python
np.testing.assert_array_equal(
    amount_ok,
    [True, True, False, True, False],
)
np.testing.assert_array_equal(
    delivered_quickly,
    [True, False, True, True, True],
)
np.testing.assert_array_equal(
    row_mask,
    [True, False, False, True, False],
)

Выберите строки и столбцы в два понятных этапа

python
selected_rows = orders[row_mask]
selected = selected_rows[:, [ORDER_ID, AMOUNT, DELIVERY_DAYS]]

assert selected.shape == (2, 3)
assert selected.dtype == orders.dtype
np.testing.assert_array_equal(
    selected,
    [
        [101, 1200, 2],
        [104, 3200, 3],
    ],
)

Первый этап выбирает наблюдения, второй — признаки. Оба используют advanced indexing и возвращают независимые буферы:

python
assert not np.shares_memory(orders, selected_rows)
assert not np.shares_memory(orders, selected)

Выборка не создает отдельный dtype для каждого столбца: весь результат сохраняет общий тип исходного ndarray. Поэтому order_id в этом примере остается float64 вместе с суммой и сроком доставки — еще одна причина перейти к табличной структуре для смешанных семантических типов.

Два этапа здесь немного длиннее, зато смысл каждой оси и ожидаемая форма остаются видимыми.

Сохраните ось признаков, если следующему шагу нужна матрица

python
amount_vector = orders[:, AMOUNT]
amount_matrix = orders[:, AMOUNT:AMOUNT + 1]

assert amount_vector.shape == (5,)
assert amount_matrix.shape == (5, 1)

Оба результата содержат одну сумму на заказ. Выбор зависит не от значений, а от контракта следующей функции.

Изменяйте источник только явно

python
scores = np.array([12.0, -3.0, 18.0])
invalid = scores < 0

cleaned = scores.copy()
cleaned[invalid] = np.nan

np.testing.assert_array_equal(scores, [12.0, -3.0, 18.0])
np.testing.assert_array_equal(cleaned, [12.0, np.nan, 18.0])

Если изменение источника действительно является частью контракта, оно должно быть видно в имени операции, параметре или документации:

python
scores[invalid] = np.nan
np.testing.assert_array_equal(scores, [12.0, np.nan, 18.0])

Замена пропуска на ноль, отрицательного значения на NaN или выброса на границу — это предметное решение, а не свойство NumPy. Урок показывает механизм изменения, но политика должна следовать из смысла данных.

Где заканчивается применимость артефакта

Артефакт намеренно работает только с непустой двумерной числовой матрицей. Он не хранит:

  • имена столбцов;
  • разные dtype для разных признаков;
  • бизнес-ключ отдельно от позиции строки;
  • причины пропусков;
  • предметную политику исправления значений.

Для однородных числовых расчетов это полезное ограничение. Для обычной аналитической таблицы с идентификаторами, категориями, датами и именованными столбцами после фазы NumPy мы перейдем к pandas.

Сломайте это

Ошибка 1. Включить правую границу среза мысленно

python
values = np.array([10, 20, 30, 40, 50])
part = values[1:4]

part содержит позиции 1, 2, 3, но не 4. Для ожидания ровно трех элементов проверьте part.shape == (3,).

Ошибка 2. Случайно удалить ось

python
one_feature = orders[:, 2]
assert one_feature.shape == (5,)

Если требуется матрица (observation, feature), используйте orders[:, 2:3] или orders[:, [2]]; оба дадут форму (5, 1), но первый результат обычно view, а второй — копия.

Ошибка 3. Использовать and или or для массивов

python
mask = (amounts >= 1000) and (amounts <= 5000)

Python пытается получить одно логическое значение всего массива. Используйте & или | и скобки вокруг каждого сравнения.

Ошибка 4. Убрать скобки вокруг сравнений

python
mask = amounts >= 1000 & amounts <= 5000

Приоритет операторов не выражает требуемую цепочку поэлементных условий. Надежная запись:

python
mask = (amounts >= 1000) & (amounts <= 5000)

Ошибка 5. Передать маску формы (row, 1) вместо (row,)

python
row_mask = np.array([[True], [False], [False], [True], [False]])
orders[row_mask]

Для выбора строк нужна одномерная маска формы (5,). Не рассчитывайте на broadcasting в boolean indexing.

Ошибка 6. Принять маску элементов за фильтр строк

python
positive = orders > 0
result = orders[positive]

result будет одномерным набором отдельных значений. Если нужно оставить строки по условию на сумму, сначала постройте orders[:, AMOUNT] > 0, получите маску (5,) и примените ее к первой оси.

Ошибка 7. Скрыто изменить источник через view

python
source = np.arange(12).reshape(3, 4)
part = source[:2]
part[:, 0] = -1

assert source[0, 0] == -1

Если изменение источника не требуется:

python
part = source[:2].copy()
assert not np.shares_memory(source, part)

Ошибка 8. Пытаться изменить источник через временную advanced-копию

python
source = np.arange(12).reshape(3, 4)
source[[0, 2]][:, 1] = -1

source не изменится. Первая операция собрала временную копию. Используйте один индекс:

python
source[[0, 2], 1] = -1

Ошибка 9. Ожидать прямоугольник от двух массивов индексов

python
result = source[[0, 2], [1, 3]]
assert result.shape == (2,)

Выбраны две пары координат. Для прямоугольника используйте:

python
result = source[np.ix_([0, 2], [1, 3])]
assert result.shape == (2, 2)

Ошибка 10. Не назвать политику NaN и бесконечностей

python
mask = amounts <= 5000

NaN будет исключен как побочный эффект сравнения, но смысл останется скрытым. Запись np.isfinite(amounts) & (amounts <= 5000) делает политику явной и исключает также бесконечности.

Ошибка 11. Забыть о dtype при присваивании

python
counts = np.array([1, 2, 3], dtype=np.int16)
counts[counts < 2] = 1.5

Тип массива не меняется автоматически ради replacement. Возможна потеря дробной части, ошибка или выход за диапазон в зависимости от операции и версии NumPy. Артефакт replace_where использует casting="same_value" и отклоняет замену, которую нельзя сохранить без изменения значения. Это прямое продолжение dtype-контракта прошлого урока.

Проверьте это

Для любой важной выборки проверяйте не только значения.

python
source_before = orders.copy()

assert row_mask.dtype == np.dtype("bool")
assert row_mask.shape == (orders.shape[0],)
assert selected.shape == (int(row_mask.sum()), 3)
assert not np.shares_memory(orders, selected)
np.testing.assert_array_equal(orders, source_before)

Полезные инварианты:

  1. число решений маски равно длине выбранной оси;
  2. число строк результата равно количеству True;
  3. число столбцов результата соответствует контракту проекции;
  4. порядок строк и столбцов предсказуем;
  5. нечисловые значения обрабатываются по явной политике;
  6. источник изменяется только в режиме, где это обещано;
  7. replacement сохраняется в существующем dtype без неожиданного преобразования.

Ключевые термины

Термин Распространенное заблуждение Точное значение
Позиционный индекс «Бизнес-идентификатор строки» Координата на оси ndarray, считающаяся от нуля или с конца
Срез «Обе границы включаются» Полуинтервал start:stop:step, в котором stop не включается
: «Любые значения массива» Сохранение всех позиций соответствующей оси
np.newaxis «Создание новых значений» Вставка оси длины один в результат индексации
Basic indexing «Любой доступ через скобки» Индексация целыми числами, срезами, ... и newaxis; массив-результат обычно является view
Advanced indexing «Более сложный, но такой же срез» Выбор массивами целых индексов или булевыми масками; при чтении создается копия
Булева маска «Одно значение True или False» Массив позиционных решений для конкретной оси или всех элементов
Маска строк «Любой bool-массив» Одномерная маска формы (row,), выбирающая позиции первой оси матрицы
View «Независимый подмассив» Объект массива, разделяющий область памяти с источником
Copy «Еще одно имя того же массива» Массив с независимым буфером данных
Индексированное присваивание «Изменение временной копии» Запись непосредственно в позиции объекта слева, например x[mask] = value
np.shares_memory «Проверка равенства значений» Точная проверка пересечения областей памяти двух массивов
np.ix_ «Обязательный синтаксис любого среза» Способ построить прямоугольный выбор по нескольким одномерным наборам индексов

Quiz · 8 вопроса

Проверьте себя

Ответьте после практики. Объяснение появится сразу после выбора.

01

Следующая функция ожидает матрицу (observation, feature). Как выбрать один столбец x с формы (8, 5), сохранив обе оси?

Срез 2:3 выбирает один признак, но сохраняет его ось. Целочисленный индекс x[:, 2] удалил бы ось признаков.

02

Матрица x имеет форму (6, 4). К первой оси применили маску формы (6,), содержащую три True. Какова форма x[mask]?

Одномерная маска первой оси оставляет три строки и сохраняет все четыре признака.

03

К матрице x формы (3, 4) применили булеву маску той же формы, в которой пять True. Что вернет x[mask]?

Маска всех элементов собирает позиции True в одномерный результат. Это не фильтрация строк.

04

Нужно исправить значения в первых ста строках, не меняя большой исходный массив source. Какой вариант фиксирует это намерение?

Базовый срез обычно разделяет память с source. Явная copy создает независимый буфер перед изменением.

05

Какое утверждение правильно описывает boolean indexing?

Advanced indexing создает копию при чтении. Индексированное присваивание является записью в объект слева и изменяет x.

06

Почему выражение x[[0, 2]][:, 1] = -1 может оставить x без изменений?

Изменение применяется к временной копии, созданной первым advanced indexing. Для источника нужен один индекс: x[[0, 2], 1] = -1.

07

Нужно получить прямоугольник из строк [0, 2] и столбцов [1, 3]. Как избежать парного выбора координат?

np.ix_ строит декартов выбор строк и столбцов. Два массива в x[[0, 2], [1, 3]] выбрали бы только пары (0, 1) и (2, 3).

08

У массива daily форма (2, 3, 4). Какова форма daily[..., np.newaxis]?

Многоточие сохраняет все существующие оси, а np.newaxis добавляет справа ось длины один.

from __future__ import annotations

import importlib.util
import json
import subprocess
import sys
import unittest
from pathlib import Path

import numpy as np

ROOT = Path(__file__).resolve().parents[1]
ARTIFACT = ROOT / "outputs" / "selection_contract.py"
EXAMPLE = ROOT / "code" / "main.py"
SPEC = importlib.util.spec_from_file_location("selection_contract", ARTIFACT)
if SPEC is None or SPEC.loader is None:
    raise RuntimeError(f"cannot load {ARTIFACT}")
SELECTION = importlib.util.module_from_spec(SPEC)
SPEC.loader.exec_module(SELECTION)


class SelectionContractTest(unittest.TestCase):
    def setUp(self) -> None:
        self.orders = np.array(
            [
                [101, 1, 1200, 2],
                [102, 3, 4500, 7],
                [103, 2, np.nan, 4],
                [104, 2, 3200, 3],
                [105, 5, 8000, 1],
            ],
            dtype=np.float64,
        )

    def test_range_mask_respects_boundaries_and_rejects_non_finite(self) -> None:
        values = np.array([5.0, 10.0, 15.0, 20.0, np.nan, np.inf, -np.inf])
        np.testing.assert_array_equal(
            SELECTION.build_range_mask(values, lower=10, upper=20),
            [False, True, True, True, False, False, False],
        )
        np.testing.assert_array_equal(
            SELECTION.build_range_mask(
                values,
                lower=10,
                upper=20,
                inclusive="neither",
            ),
            [False, False, True, False, False, False, False],
        )

    def test_range_mask_supports_one_sided_bounds(self) -> None:
        values = np.array([1, 2, 3, 4])
        np.testing.assert_array_equal(
            SELECTION.build_range_mask(values, lower=3),
            [False, False, True, True],
        )
        np.testing.assert_array_equal(
            SELECTION.build_range_mask(values, upper=2, inclusive="neither"),
            [True, False, False, False],
        )

    def test_range_mask_rejects_invalid_contract(self) -> None:
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "cannot exceed"):
            SELECTION.build_range_mask([1, 2, 3], lower=5, upper=2)
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "must be finite"):
            SELECTION.build_range_mask([1, 2, 3], lower=np.nan)
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "1-dimensional"):
            SELECTION.build_range_mask([[1, 2], [3, 4]], lower=1)

    def test_select_rows_preserves_order_shape_and_requested_columns(self) -> None:
        mask = np.array([True, False, False, True, False])
        selected = SELECTION.select_rows(
            self.orders,
            mask,
            columns=[0, 2, 3],
        )
        self.assertEqual(selected.shape, (2, 3))
        np.testing.assert_array_equal(
            selected,
            [[101, 1200, 2], [104, 3200, 3]],
        )

    def test_select_rows_accepts_negative_column_indices(self) -> None:
        selected = SELECTION.select_rows(
            self.orders,
            [True, False, False, False, True],
            columns=[0, -1],
        )
        np.testing.assert_array_equal(selected, [[101, 2], [105, 1]])

    def test_selected_result_is_independent_from_source(self) -> None:
        source = self.orders.copy()
        selected = SELECTION.select_rows(
            source,
            [True, False, False, True, False],
        )
        self.assertFalse(np.shares_memory(source, selected))
        selected[0, 0] = 999
        self.assertEqual(source[0, 0], 101)

    def test_select_rows_requires_one_boolean_decision_per_row(self) -> None:
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "bool dtype"):
            SELECTION.select_rows(self.orders, [1, 0, 0, 1, 0])
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "row mask shape"):
            SELECTION.select_rows(self.orders, [[True], [False], [False], [True], [False]])

    def test_select_rows_rejects_duplicate_or_out_of_range_columns(self) -> None:
        mask = [True, False, False, True, False]
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "duplicates"):
            SELECTION.select_rows(self.orders, mask, columns=[0, -4])
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "out of bounds"):
            SELECTION.select_rows(self.orders, mask, columns=[4])

    def test_replace_where_copies_by_default(self) -> None:
        source = np.array([[1, 2], [3, 4]], dtype=np.int16)
        mask = np.array([[False, True], [True, False]])
        result = SELECTION.replace_where(source, mask, 0.0)
        np.testing.assert_array_equal(source, [[1, 2], [3, 4]])
        np.testing.assert_array_equal(result, [[1, 0], [0, 4]])
        self.assertFalse(np.shares_memory(source, result))

    def test_replace_where_can_change_source_explicitly(self) -> None:
        source = np.array([1.0, 2.0, 3.0])
        result = SELECTION.replace_where(
            source,
            source > 1,
            np.nan,
            in_place=True,
        )
        self.assertIs(result, source)
        np.testing.assert_array_equal(source, [1.0, np.nan, np.nan])

    def test_replace_where_checks_mask_and_replacement_contracts(self) -> None:
        source = np.array([1, 2, 3], dtype=np.int16)
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "bool dtype"):
            SELECTION.replace_where(source, [0, 1, 0], 0)
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "mask shape"):
            SELECTION.replace_where(source, [True], 0)
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "cannot be stored"):
            SELECTION.replace_where(source, [False, True, False], 1.5)
        with self.assertRaisesRegex(SELECTION.SelectionContractError, "requires a NumPy"):
            SELECTION.replace_where([1, 2, 3], [False, True, False], 0, in_place=True)

    def test_memory_report_exposes_shape_and_buffer_differences(self) -> None:
        report = SELECTION.memory_report(self.orders)
        self.assertEqual(report["source_shape"], [5, 4])
        self.assertEqual(report["basic_slice_shape"], [2, 4])
        self.assertTrue(report["basic_slice_shares_memory"])
        self.assertEqual(report["advanced_selection_shape"], [2, 4])
        self.assertFalse(report["advanced_selection_shares_memory"])
        self.assertEqual(report["column_vector_shape"], [5])
        self.assertEqual(report["column_matrix_shape"], [5, 1])

    def test_selection_report_connects_mask_shape_and_memory_contracts(self) -> None:
        report = SELECTION.build_selection_report(
            self.orders,
            filter_column=2,
            lower=1000,
            upper=5000,
            columns=[0, 2, 3],
        )
        self.assertEqual(report["axis_contract"], ["observation", "feature"])
        self.assertEqual(report["mask_shape"], [5])
        self.assertEqual(report["selected_count"], 3)
        self.assertEqual(report["excluded_non_finite"], 1)
        self.assertEqual(report["selected_shape"], [3, 3])
        self.assertFalse(report["shares_memory"])
        self.assertEqual(report["selected"][0], [101, 1200, 2])

    def test_json_null_is_treated_as_missing_numeric_value(self) -> None:
        matrix = [[1, 10], [2, None], [3, 30]]
        report = SELECTION.build_selection_report(
            matrix,
            filter_column=1,
            lower=0,
            upper=100,
        )
        self.assertEqual(report["source_dtype"], "float64")
        self.assertEqual(report["mask"], [True, False, True])
        self.assertEqual(report["excluded_non_finite"], 1)

    def test_cli_emits_auditable_report(self) -> None:
        result = subprocess.run(
            [
                sys.executable,
                ARTIFACT,
                "--lower",
                "1000",
                "--upper",
                "5000",
            ],
            check=False,
            capture_output=True,
            text=True,
        )
        self.assertEqual(result.returncode, 0, result.stderr)
        report = json.loads(result.stdout)
        self.assertEqual(report["source_shape"], [5, 4])
        self.assertEqual(report["selected_shape"], [3, 3])
        self.assertEqual(report["excluded_non_finite"], 1)
        self.assertFalse(report["shares_memory"])

    def test_cli_rejects_invalid_selection_without_traceback(self) -> None:
        result = subprocess.run(
            [
                sys.executable,
                ARTIFACT,
                "--filter-column",
                "8",
            ],
            check=False,
            capture_output=True,
            text=True,
        )
        self.assertEqual(result.returncode, 2)
        self.assertIn("out of bounds", result.stderr)
        self.assertNotIn("Traceback", result.stderr)

    def test_main_example_runs(self) -> None:
        result = subprocess.run(
            [sys.executable, EXAMPLE],
            check=False,
            capture_output=True,
            text=True,
        )
        self.assertEqual(result.returncode, 0, result.stderr)
        self.assertIn("selected shape: (2, 3)", result.stdout)
        self.assertIn("shares memory: False", result.stdout)


if __name__ == "__main__":
    unittest.main()

from __future__ import annotations

import argparse
import json
import operator
from pathlib import Path
from typing import Any

import numpy as np


class SelectionContractError(ValueError):
    """Raised when an array selection contract is invalid."""


def _as_numeric_array(
    values: object,
    *,
    name: str,
    ndim: int | None = None,
) -> np.ndarray:
    try:
        array = np.asarray(values)
    except ValueError as error:
        raise SelectionContractError(f"{name} must be rectangular") from error

    if ndim is not None and array.ndim != ndim:
        raise SelectionContractError(
            f"{name} must be {ndim}-dimensional, got shape {array.shape}"
        )
    if array.ndim == 0:
        raise SelectionContractError(f"{name} must contain an array, not a scalar")
    if 0 in array.shape:
        raise SelectionContractError(f"{name} must not have an empty axis")

    if array.dtype.kind == "O":
        try:
            array = np.asarray(values, dtype=np.float64)
        except (TypeError, ValueError) as error:
            raise SelectionContractError(f"{name} must contain numeric values") from error

    if array.dtype.kind not in "iuf":
        raise SelectionContractError(
            f"{name} must contain numeric values, got dtype {array.dtype}"
        )
    return array


def as_numeric_matrix(values: object) -> np.ndarray:
    """Return a non-empty two-dimensional numeric array.

    JSON ``null`` values are converted to ``NaN`` and therefore require a floating dtype.
    """

    return _as_numeric_array(values, name="matrix", ndim=2)


def _validate_bounds(
    lower: float | None,
    upper: float | None,
) -> tuple[float | None, float | None]:
    normalized_lower = None if lower is None else float(lower)
    normalized_upper = None if upper is None else float(upper)

    for name, value in (("lower", normalized_lower), ("upper", normalized_upper)):
        if value is not None and not np.isfinite(value):
            raise SelectionContractError(f"{name} bound must be finite")
    if (
        normalized_lower is not None
        and normalized_upper is not None
        and normalized_lower > normalized_upper
    ):
        raise SelectionContractError("lower bound cannot exceed upper bound")
    return normalized_lower, normalized_upper


def build_range_mask(
    values: object,
    *,
    lower: float | None = None,
    upper: float | None = None,
    inclusive: str = "both",
) -> np.ndarray:
    """Build a one-dimensional mask and always reject non-finite values."""

    if inclusive not in {"both", "left", "right", "neither"}:
        raise SelectionContractError(
            "inclusive must be both, left, right, or neither"
        )
    normalized_lower, normalized_upper = _validate_bounds(lower, upper)
    array = _as_numeric_array(values, name="filter values", ndim=1)

    mask = np.isfinite(array)
    if normalized_lower is not None:
        lower_mask = (
            array >= normalized_lower
            if inclusive in {"both", "left"}
            else array > normalized_lower
        )
        mask &= lower_mask
    if normalized_upper is not None:
        upper_mask = (
            array <= normalized_upper
            if inclusive in {"both", "right"}
            else array < normalized_upper
        )
        mask &= upper_mask
    return mask


def _normalize_index(index: object, length: int, *, name: str) -> int:
    if isinstance(index, (bool, np.bool_)):
        raise SelectionContractError(f"{name} must be an integer, not bool")
    try:
        normalized = operator.index(index)
    except TypeError as error:
        raise SelectionContractError(f"{name} must be an integer") from error
    if normalized < -length or normalized >= length:
        raise SelectionContractError(
            f"{name} {normalized} is out of bounds for axis of length {length}"
        )
    return normalized % length


def _normalize_columns(
    columns: object | None,
    feature_count: int,
) -> tuple[int, ...]:
    if columns is None:
        return tuple(range(feature_count))

    raw = np.asarray(columns)
    if raw.ndim != 1 or raw.size == 0:
        raise SelectionContractError("columns must be a non-empty one-dimensional list")
    normalized = tuple(
        _normalize_index(value, feature_count, name="column index") for value in raw
    )
    if len(set(normalized)) != len(normalized):
        raise SelectionContractError("columns must not contain duplicates")
    return normalized


def _validate_row_mask(row_mask: object, row_count: int) -> np.ndarray:
    mask = np.asarray(row_mask)
    if mask.dtype.kind != "b":
        raise SelectionContractError(
            f"row mask must have bool dtype, got {mask.dtype}"
        )
    if mask.shape != (row_count,):
        raise SelectionContractError(
            f"row mask shape {mask.shape} does not match row axis ({row_count},)"
        )
    return mask


def select_rows(
    matrix: object,
    row_mask: object,
    *,
    columns: object | None = None,
) -> np.ndarray:
    """Select rows and columns, preserving their order and returning an independent array."""

    array = as_numeric_matrix(matrix)
    mask = _validate_row_mask(row_mask, array.shape[0])
    normalized_columns = _normalize_columns(columns, array.shape[1])

    selected_rows = array[mask]
    selected = selected_rows[:, normalized_columns]
    result = selected.copy()
    if np.shares_memory(array, result):
        raise SelectionContractError("selected result unexpectedly shares memory")
    return result


def replace_where(
    values: np.ndarray,
    mask: object,
    replacement: object,
    *,
    in_place: bool = False,
) -> np.ndarray:
    """Replace positions selected by a same-shape boolean mask.

    The replacement must survive conversion to the array dtype without changing value.
    """

    array = _as_numeric_array(values, name="values")
    if in_place and array is not values:
        raise SelectionContractError("in_place=True requires a NumPy ndarray")

    boolean_mask = np.asarray(mask)
    if boolean_mask.dtype.kind != "b":
        raise SelectionContractError(
            f"replacement mask must have bool dtype, got {boolean_mask.dtype}"
        )
    if boolean_mask.shape != array.shape:
        raise SelectionContractError(
            f"replacement mask shape {boolean_mask.shape} does not match values {array.shape}"
        )

    replacement_array = np.asarray(replacement)
    if replacement_array.ndim != 0:
        raise SelectionContractError("replacement must be a scalar")
    try:
        converted = replacement_array.astype(
            array.dtype,
            casting="same_value",
            copy=False,
        )
    except (TypeError, ValueError, OverflowError) as error:
        raise SelectionContractError(
            f"replacement {replacement!r} cannot be stored unchanged in dtype {array.dtype}"
        ) from error

    result = array if in_place else array.copy()
    result[boolean_mask] = converted.item()
    return result


def memory_report(matrix: object) -> dict[str, Any]:
    """Demonstrate shape and memory differences for common indexing forms."""

    array = as_numeric_matrix(matrix)
    take_count = min(2, array.shape[0])
    basic_slice = array[:take_count]
    advanced_selection = array[np.arange(take_count)]
    column_vector = array[:, 0]
    column_matrix = array[:, 0:1]
    return {
        "source_shape": list(array.shape),
        "basic_slice_shape": list(basic_slice.shape),
        "basic_slice_shares_memory": bool(np.shares_memory(array, basic_slice)),
        "advanced_selection_shape": list(advanced_selection.shape),
        "advanced_selection_shares_memory": bool(
            np.shares_memory(array, advanced_selection)
        ),
        "column_vector_shape": list(column_vector.shape),
        "column_matrix_shape": list(column_matrix.shape),
    }


def build_selection_report(
    matrix: object,
    *,
    filter_column: object,
    lower: float | None = None,
    upper: float | None = None,
    inclusive: str = "both",
    columns: object | None = None,
) -> dict[str, Any]:
    """Build a JSON-serializable audit of one row-selection contract."""

    array = as_numeric_matrix(matrix)
    normalized_filter_column = _normalize_index(
        filter_column,
        array.shape[1],
        name="filter column",
    )
    normalized_columns = _normalize_columns(columns, array.shape[1])
    filter_values = array[:, normalized_filter_column]
    mask = build_range_mask(
        filter_values,
        lower=lower,
        upper=upper,
        inclusive=inclusive,
    )
    selected = select_rows(array, mask, columns=normalized_columns)
    return {
        "axis_contract": ["observation", "feature"],
        "source_shape": list(array.shape),
        "source_dtype": str(array.dtype),
        "filter_column": normalized_filter_column,
        "columns": list(normalized_columns),
        "mask_shape": list(mask.shape),
        "mask": mask.tolist(),
        "selected_count": int(mask.sum()),
        "excluded_non_finite": int((~np.isfinite(filter_values)).sum()),
        "selected_shape": list(selected.shape),
        "selected_dtype": str(selected.dtype),
        "selected": selected.tolist(),
        "shares_memory": bool(np.shares_memory(array, selected)),
        "memory_demo": memory_report(array),
    }


def parse_json(raw: str, *, name: str) -> Any:
    try:
        return json.loads(raw)
    except json.JSONDecodeError as error:
        raise SelectionContractError(f"{name} must be valid JSON") from error


def main() -> int:
    parser = argparse.ArgumentParser(
        description="Audit NumPy row, column, mask, shape, and memory selection"
    )
    parser.add_argument(
        "--matrix",
        default=(
            "[[101, 1, 1200, 2], [102, 3, 4500, 7], "
            "[103, 2, null, 4], [104, 2, 3200, 3], [105, 5, 8000, 1]]"
        ),
    )
    parser.add_argument("--filter-column", type=int, default=2)
    parser.add_argument("--lower", type=float)
    parser.add_argument("--upper", type=float)
    parser.add_argument(
        "--inclusive",
        choices=("both", "left", "right", "neither"),
        default="both",
    )
    parser.add_argument("--columns", default="[0, 2, 3]")
    parser.add_argument("--output", type=Path)
    args = parser.parse_args()

    try:
        matrix = parse_json(args.matrix, name="matrix")
        columns = parse_json(args.columns, name="columns")
        report = build_selection_report(
            matrix,
            filter_column=args.filter_column,
            lower=args.lower,
            upper=args.upper,
            inclusive=args.inclusive,
            columns=columns,
        )
    except SelectionContractError as error:
        parser.exit(2, f"selection-contract: {error}\n")

    text = json.dumps(report, ensure_ascii=False, indent=2) + "\n"
    if args.output:
        args.output.write_text(text, encoding="utf-8")
    else:
        print(text, end="")
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
