# Рубрика персональной карты маршрута

Рубрика проверяет качество рассуждения. Она не оценивает престиж маршрута и не заменяет
диагностику Python, SQL и статистики.

## Обязательные критерии

Карта готова только когда выполнены все пункты.

- [ ] Описаны минимум три конкретные рабочие задачи, а не темы или библиотеки.
- [ ] Для каждой задачи указан получатель результата.
- [ ] Для каждой задачи названо решение или действие, которое изменится.
- [ ] Для каждой задачи указан проверяемый артефакт.
- [ ] Критерии качества позволяют другому человеку проверить результат.
- [ ] Названа цена хотя бы одной содержательной ошибки.
- [ ] Основной маршрут обоснован ближайшим рабочим артефактом.
- [ ] Интерес к маршруту отделен от текущей готовности и стартовой фазы.

## Критерии сильной карты

- [ ] Для неоднозначной задачи записан уточняющий вопрос, а не принудительная категория.
- [ ] Дополнительный маршрут отражает реальную зависимость или вторую ответственность.
- [ ] Названия вакансий проверены по обязанностям и результатам работы, а не приняты на веру.
- [ ] Объяснено, почему соседний маршрут сейчас не является основным.
- [ ] Указано наблюдаемое условие пересмотра карты.
- [ ] Краткое обоснование понятно без названий библиотек и должностей.

## Решение

- **Готово:** выполнены все обязательные критерии; сильные критерии используются для доработки.
- **Нужна правка:** не выполнен хотя бы один обязательный критерий.
- **Недостаточно контекста:** решение или артефакт нельзя назвать без ответа заказчика; сначала сформулируйте уточняющий вопрос.
