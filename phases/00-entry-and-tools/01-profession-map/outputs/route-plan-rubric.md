# Проверка персональной карты учебного маршрута

Здесь проверяется логика выбора, а не престиж направления или текущий уровень знаний.

## Карта готова, если

- [ ] Разобраны две учебные ситуации: для каждой названы решение, рабочий результат,
  направление и причина выбора.
- [ ] Записаны две возможные рабочие задачи.
- [ ] Одна задача подробно связана с получателем, решением и ожидаемым результатом.
- [ ] Основной маршрут выбран по желаемой работе, а не по названию должности или
  инструменту.
- [ ] Назван хотя бы один вопрос, на который пока не хватает ответа.
- [ ] Выбор маршрута не используется как причина пропустить общее ядро курса.

## Сильная карта дополнительно показывает

- [ ] Какое соседнее направление или зависимость поможет получить хороший результат.
- [ ] Какие обязанности искать в вакансиях независимо от названия должности.
- [ ] По каким признакам можно проверить качество результата и какая ошибка опаснее всего.
- [ ] Какой следующий шаг сделать и при каком изменении карту стоит пересмотреть.

## Итог проверки

- **Готово:** все обязательные пункты выполнены.
- **Нужна правка:** хотя бы в одном обязательном пункте отсутствует связь между задачей,
  решением, результатом и маршрутом.
- **Недостаточно контекста:** направление нельзя выбрать без ответа на уточняющий вопрос;
  сначала запишите этот вопрос.
