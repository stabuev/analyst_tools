# Персональная карта маршрута

Этот шаблон помогает выбрать последовательность обучения по ближайшим рабочим решениям,
а не по названию должности или списку библиотек.

## Как использовать

1. Заполните минимум три задачи, которые хотите уметь решать в ближайшей работе.
2. Не указывайте инструменты вместо результата: «изучить pandas» не является задачей.
3. Выберите основной маршрут по ближайшему артефакту, дополнительный — по важной зависимости.
4. Проверьте карту по `route-plan-rubric.md`.
5. Пересмотрите карту после изменения рабочей цели, а не после знакомства с новым инструментом.
