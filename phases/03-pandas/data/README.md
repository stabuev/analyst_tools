# Учебные данные фазы 03

В каталоге `tiny/` лежат три небольшие совместимые таблицы для ручного расчета:
`users.csv`, `orders.csv` и `order_items.csv`. Они намеренно содержат пропуски,
несогласованные строки, смешанные часовые пояса, удаленного пользователя и join
одного заказа с несколькими позициями.

Источник правды для схемы, grain, ключей и известных дефектов — `contract.json`.
`tiny/manifest.json` фиксирует число строк и SHA-256 каждого CSV.

Набор воспроизводится без pandas и внешних данных:

```bash
python3 phases/03-pandas/data/generate_tiny.py
```

Генератор обязан создавать байт-в-байт те же файлы. Это проверяется репозиторным тестом.
