# Глоссарий

Глоссарий фиксирует значения терминов именно в контексте курса.

- [Термины](terms.md)
- [Распространенные заблуждения](myths.md)

Термин добавляется сюда, если используется в нескольких фазах или имеет несколько
конфликтующих трактовок в индустрии. Локальные понятия одного урока остаются в его разделе
«Ключевые термины».
